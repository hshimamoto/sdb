Small DataBase
==============

Usage
-----

database file is `database.sdb` in current directory.

```
$ sdb.rb <command> [opt]
```

### command

#### dump
Dump database. Separated by `tab`.

#### edit <file>
Edit database through `vim <file>`.
The 1st line is header.

Required Package
----------------
- ruby
